tools fix
This is the way to implement the fix for the tools directory

cd $NWCHEM_TOP/src/tools
rm -rf ga-5.6.5* build install
wget https://github.com/edoapra/ga/releases/download/v5.6.5/ga-5.6.5.tar.gz
make
cd ..
make link
